You are designing ONLY the UI for the Boundier Chrome extension.
No backend code. No scraping logic. No scoring logic. Design only.

==================================================
PHILOSOPHY
==================================================
Visual language must match the Boundier prototype:
- "Liquid glass" panels
- Cold blue tech aesthetic
- Clean, quiet, minimal
- Zero dopamine-bait visuals

The extension UI should feel like a diagnostic instrument,
not entertainment and not social media.

==================================================
GLOBAL DESIGN SYSTEM
==================================================

FONT:
  - Cooper Hewitt only
  - Light / Regular / Medium / Semibold
  - No bold paragraphs. Titles only.

COLORS:
  #000543 (primary dark)
  #0038FF (accent blue)
  #FFFFFF (pure white)
  rgba(255,255,255,0.07) (light glass)
  rgba(0,5,67,0.32) (dark glass tint)

GLASS PANEL STYLE:
  background: rgba(0,5,67,0.32)
  backdrop-filter: blur(12px)
  border-radius: 16px
  box-shadow: 0 4px 20px rgba(0,0,0,0.25)
  stroke: 1px solid rgba(255,255,255,0.10)

MOTION:
  Slide-ins: 220ms spring
  Fade: 180ms
  Micro-parallax: panel shifts 2–4px on scroll

SPACING:
  Major section padding: 20px
  Internal component padding: 16px
  Element gaps: 12px

Icon style:
  geometric, monochrome, no gradients
